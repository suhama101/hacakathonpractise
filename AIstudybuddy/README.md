# AI Study Buddy Pakistan 📚

Pakistan ke students ke liye AI-powered personal tutor.

## Problem
Pakistan mein 70% students private tutor afford nahi kar sakte.

## Solution
AI Study Buddy — har subject ka expert, 24/7 available, bilkul free.

## Features
- 💬 AI Chatbot — koi bhi sawaal poochho
- 📸 Image Analysis — math problem photo se solve karo
- 📚 Multiple Subjects — Math, Physics, Chemistry, CS, Biology
- 💾 History Save — purane conversations yaad rehte hain
- 🌐 Urdu/English — dono languages support

## Tech Stack
- Python + Streamlit
- Groq AI (Llama 3.3)
- Meta Llama Vision

## How to Run
pip install -r requirements.txt
streamlit run app.py
